---
title: '环境变量'
description: '环境变量'
---

Next-Forge具有一个可组合的环境变量系统，该系统允许您为每个软件包以类型安全的方式定义和验证环境变量，然后将它们组合到该应用程序的单个环境变量文件中。

# 概述

类型安全由 @t3-oss/env-nextjs 提供，它为所有环境变量提供运行时验证和自动补全功能。每个包都可以在 keys.ts 文件中定义自己的环境变量，该文件导出一个包含所有已验证环境变量的 keys 对象，这些变量被分为服务器端和客户端变量。

这些 keys 随后会被组合到应用程序根目录下的 env.ts 文件中，形成一个单一的环境变量文件。
